# УП ПО АИП
Репозиторий с кодом практических заданий по курсу.

## Структура веток:
- `main` — общая информация (этот файл).
- `day1` — код для Day 1 (веб-калькулятор).
- `day2` — код для Day 2 (тестирование калькулятора).
- `day3` — код для Day 3 (тестирование функций).
- `day4` — код для Day 4 (интеграционное тестирование REST API).
